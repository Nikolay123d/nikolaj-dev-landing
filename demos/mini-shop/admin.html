Demo admin / není veřejná administrace

Public admin není součástí obchodu

Tato stránka záměrně neobsahuje funkční veřejnou administraci. Static e-shop nemá bezpečný backend login; lokální editor je pouze demo nástroj pro vlastníka.

Pro úpravu produktů použijte admin-local/: změny se uloží v localStorage, poté exportujete products.json a ručně nahradíte soubor v projektu nebo na hostingu.
